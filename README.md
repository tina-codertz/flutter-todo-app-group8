
# flutter_todo_app_group8

## How to Run the App

| S/N | Details |
|-----|--------|
| 1 | Clone the repository to your machine: `git clone https://github.com/tina-codertz/flutter-todo-app-group8.git` |
| 2 | Navigate to the project folder: `cd flutter-todo-app-group8` |
| 3 | Install the required Flutter dependencies: `flutter pub get` |
| 4 | Run the app: `flutter run` |
